Detailed Description

Point at infinity on an elliptic curve, that is, the neutral element of the abelian group of points; it supports additive infix notation for group operations, as well as multiplication by integers.

Note
This implementation is independent of the representation used for finite points.

Return the sum of self and other.

The infix operator + calls this method if self is the left summand.

The point at infinity is the neutral element of an elliptic curve. Hence the returned value will always be other.
