Detailed Description

Base class for nodes of CallGraphs.

The class provides the mechanisms that ensure correct ownership of nodes: all nodes belong to a unique CallGraph that manages all edge information. If a node wants to know its neighbors, it asks its owning CallGraph (for example via CallGraph._caller() and the like).

When the node is removed, the graph invokes _invalidate(). Afterwards, _assert_valid() raises a ValueError. New node types should inherit from this class.
