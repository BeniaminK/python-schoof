Detailed Description

A simple parser for "separated value" (SV) files that makes sure that every line of the file is yielded in only one instance operating on it.

Use this class, for instance, to have multiple instances of a program operate on the same parameter list. The class makes sure that every line is read by only one instance of the program.

The different instances of ParallelParser synchronize their operation through a shared text file. File system locking prevents concurrent reads and writes. Since this works even on networked file systems, parser instances can in this way communicate across a network. The communication file is named file.pipe, where file is the name of the parsed file.

To iterate over a file foo.txt line by line, you could use the following code:

1 for line in ParallelParser( "foo.txt" ):
2  # Do something with line

Now suppose you have a computation intensive algorithm f(x,y) and you want to apply it once to the parameter set of every line in the parameter list in param.lst. Thus you write a program like

1 for params in ParallelParser( "param.lst", "," ):
2  f(*params) # Expand the tuple params

If you start, say, two instances of the program and the contents of param.lst are

1 1,2
2 3,4
3 5,6

then one instance will receive the tuple (1,2) and the other (3,4). The final tuple (5,6) will be read by whichever instance finishes first.

Note
This class is designed for use cases where each yielded line takes a long time to process. Synchronization by file locking probably involves a large overhead, so fast processing with many parsers at the same time will scale badly.

Parameters
fileThe name of the file to be parsed. It als serves as the first part of the name of the shared communication file file.pipe.
separatorThe parser splits the entries in each line of file by separator. For example, to retrieve a the entries of a comma separated value (CSV) file as a tuple, use "," as separator. The line "1,2,3" then yields the tuple (1,2,3); with a space as separator, the parser yields a tuple with one entry: the string "1,2,3".

def support.running.ParallelParser.__iter__ (   self)

Interface method to support line by line iteration with the for statement.

Every line is split with the separator provided to __init__(), the method thus returns a tuple of contents. For example, the line "1,2,3" will yield (1,2,3) if the separator is ",".

The parser skips empty lines and comment lines, that is, lines that start with zero or more spaces followed by a hash '#'.

Returns
A tuple of the line contents, split by separator.
