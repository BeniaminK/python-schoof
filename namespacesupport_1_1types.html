Detailed Description

A template meta-class system.

Template meta-classes offer high modularity and natural syntax for mathematical objects; for instance, see rings.polynomials.naive.Polynomials or fields.finite.naive.FiniteField

def support.types.template (   parameters)

Factory function to create template meta-classes.

The function arguments are the names of the template parameters. After specialization, the bound parameters are available as class attributes of the given names.

Usage example:

1 # Create a class template C with parameters "a" and "b"
2 class C( metaclass=template("a", "b") ):
3  pass
4 
5 # Instantiate the template with values 23 and 42; Cp is a class object.
6 Cp = C(23, 42)
7 isinstance(Cp, type) == True # This holds
8 Cp.a == 23
9 Cp.b == 42
10 
11 # Construct Cp objects
12 x = Cp()
13 type(x) is Cp # This also holds
