Detailed Description

A template meta-class prototype; use the function template() to create actual meta-class types.

Outline of usage and effects:

1 # Create a template meta-class 'T' with parameters "a" and "b".
2 # The meta-class can be used for _any_ class.
3 T = template("a", "b")
4 
5 # Create a class template with parameters "a" and "b" (these come from T)
6 # The type of C is T<a,b>
7 class C(metaclass=T):
8  def __str__(self): return "C"
9 
10 # Partially specialize C by setting parameter a; this yields a class
11 # template D with the single parameter "b"; type(D) is T<a=4,b> and
12 # is a subtype of T<a,b>
13 D = C(4)
14 
15 # Create a subclass template E with parameters "a" and "b"; use this to
16 # re-implement methods from the class template C
17 class E(C):
18  def __str__(self): return "E"
19 # Parameters can be bound while subclassing template classes
20 class F(C, b=2):
21  def __str__(self): return "F with b=2"
