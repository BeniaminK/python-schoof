Detailed Description

Auxiliary functions for computations related to prime numbers.

The algorithms and implementations are kept as simple as possible. They are not meant for high performance computing, but for instructive purposes.

def support.primes.inverse_primorial (   n,
  shunned = 0 
)

Return the smallest prime p such that the product of all primes up to p (inclusive) is at least n.

For example, inverse_primorial(30) is 5, and inverse_primorial(31) is 7.

Note
This function uses primes_range() to obtain a list of primes. See the notes there for use case limitations.
Parameters
nThe number object that the product must exceed in size. The object must have an integer interpretation.
Returns
The prime p such that the product of all primes up to p (inclusive) is at least n; if n is too small (less than 2) the result is 2.

def support.primes.primes_range (   lower_bound,
  upper_bound

Return a list of all primes in the range [lower_bound, upper_bound).

Note
The function returns an actual list, not a proper range object. Furthermore, it computes the complete list of primes up to upper_bound on each call. These characteristics make it badly suited for anything involving large ranges or large primes.
Parameters
lower_boundThe minimum size for returned primes; typically this is an integer, but any number object with an integer interpretation works.
upper_boundThe strict upper bound for returned primes: all returned primes are strictly less than this number.
