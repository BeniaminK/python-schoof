Detailed Description

A base class for elements of commutative rings that provides default operator overloading.

For example, subtraction is defined as addition of the additive inverse. Derived classes therefore only have to implement the following operations to support the complete operator set:

  • __bool__(): Zero testing (True if not zero)
  • __eq__(): Equality testing with the == operator (True if equal)
  • __add__(): Addition with the + operator; self is the left summand
  • __neg__(): Negation with the - unary minus operator (the additive inverse)
  • __mul__(): Multiplication with the * operator; self is the left factor
  • __divmod__(): Division with remainder; self is the dividend (left element)

The following operations are implemented in terms of above operations using the field axioms:

  • __neq__(): Inequality testing with the != operator
  • __radd__(): Addition with the + operator; self is the right summand
  • __sub__(): Subtraction with the - operator; self is the minuend (left element)
  • __rsub__(): Subtraction with the - operator; self is the subtrahend (right element)
  • __rmul__(): Multiplication with the * operator; self is the right factor
  • __rdivmod__(): Division with remainder using divmod(); self is the divisor (right element)
  • __floordiv__(): Remainder ignoring division with the // operator; self is the dividend (left element)
  • __rfloordiv__(): Remainder ignoring division with the // operator; self is the divisor (right element)
  • __mod__(): Remainder of self modulo other with the % operator
  • __rmod__(): Remainder of other modulo self with the % operator
  • __pow__(): Exponentiation with integers

Return self taken to the n-th power.

The infix operator ** calls this method; for example:

1 result = self ** n
Note
The implementation uses the most naive by-the-book method for exponentiation: the element is multiplied n-1 times with itself. This is slow (and might skin your cat!). However, the purpose of this code is to be easy to understand, not fast.
Parameters
nThe exponent; it is expected to be a non-negative integer type. Negative integers and floats are unsupported.
